### 6. 权限控制（RBAC）❌ 未完成 (20%)

**已有基础**:
- User 模型有 `role` 字段
- JWT 包含 role 信息
- 中间件提取 role

**缺失内容**:
- ❌ 权限表（permissions）
- ❌ 角色-权限关联表（role_permissions）
- ❌ RBAC 中间件
- ❌ API 权限控制
- ❌ 管理员 API

**建议实现**:

```sql
-- 权限表
CREATE TABLE permissions (
    id UUID PRIMARY KEY,
    name VARCHAR(100) NOT NULL,  -- skill:create, skill:delete, user:manage
    description TEXT
);

-- 角色权限关联
CREATE TABLE role_permissions (
    role VARCHAR(20) REFERENCES users(role),
    permission_id UUID REFERENCES permissions(id),
    PRIMARY KEY (role, permission_id)
);
```

**API 中间件示例**:
```rust
// 需要管理员权限
async fn admin_only(AuthUser(user): AuthUser) -> Result<...> {
    if user.role != "admin" {
        return Err(ApiError::Forbidden);
    }
    // ...
}
```
